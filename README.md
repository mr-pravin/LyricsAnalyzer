# AI_Lyric_Generator

## Description
Generates original song lyrics using LSTM neural networks. Users can input mood, theme, or keyword to produce new lyrics.

## Features
- Generates song lyrics in multiple genres
- Accepts user input for theme and mood
- Provides downloadable lyrics as .txt files
- Web interface built with Flask

## Tech Stack
- Python, TensorFlow/Keras
- Flask for web interface
- NLTK for text preprocessing
